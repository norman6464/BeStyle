# BeStyle

## **【開発背景と解決できること】**

マッチングアプリやSNSにおいて、「テキスト上の印象」と「実際の人物像」の乖離は大きな課題です。また、理想の姿はあるものの、日々のコミュニケーションでどう表現すればいいか悩む人は少なくありません。

本アプリは、ユーザーが設定した「理想の人物像」を軸に、AIがコミュニケーションの質を可視化。以下のプロセスで自己実現をサポートします。

1. **オープン投稿：** なりたい自分を公開し、共感したユーザーとマッチング。
2. **チャット分析：** 実際のやり取りをAIが読み取り、「理想の像に沿った振る舞いができているか」を評価。
3. **改善フィードバック：** AIが客観的なギャップを指摘。コミュニケーションの癖を修正し、一貫性のある自己表現を身につけることができます。
